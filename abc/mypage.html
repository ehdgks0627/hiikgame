완료되었습니다.

◎ HIIK 충전

1 ) HIIK 으로 충전

1. [HIIK GAME]-[마이페이지]-[회원정보 수정]-[지갑주소] 복사

2. 출금하려는 페이지(ex.거래소)의 [지급]-[HIIK]-[출금주소] 에 입력 후 출금

3. HIIK Token 충전완료 및 보유량 확인

2 ) 국가화폐 (pay) 로 충전

1. [PAY] 에서 로그인 후 원화로 국가화폐 충전

2. [HIIK GAME]-[지갑]-[HIIK 충전하기] 에서 수량과 충전화폐를 선택하여 충전

3. 국가화폐로 HIIK 충전완료 및 보유량 확인

◎ HIIK 환전

1 ) HIIK 출금

1. [HIIK GAME]-[지갑]-[HIIK 전환하기] 에서 수량 선택

2. [출금주소] 에 보내려는 지갑주소 입력

2 ) 국가화폐 (pay) 로 환전

1. [HIIK GAME]-[지갑]-[HIIK 환전하기] 에서 수량과 환전화폐를 선택하여 환전

2. [HIIK PAY] 에서 로그인 후 보유량 확인

◎ POINT 전환

-HIIK을 포인트로 전환 후 게임에 참여할 수 있습니다.

1) 포인트 충전하기or전환하기 선택

2) 원하는 수량 입력

3) 충전or전환 완료
